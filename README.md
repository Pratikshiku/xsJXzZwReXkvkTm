## Supermarket Inventory Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0c8c112e1cb649ff80b2ca590a10f8a5/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Layui`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Super Administrator, mainly responsible for goods and supplier management, Purchasing Personnel, Sales Personnel, Regular User



**Main Functions:**
1. Login
2. Product Management
3. Category Management
4. Warning Products
5. Supermarket Returns
6. Supply Chain Management
7. Product Inbound
8. Product Outbound
9. Sales Orders
10. Sales Returns
11. Customer Management
12. Salary Management
13. Purchase Expenses
14. Financial Analysis
15. Department Management
16. User Management
17. Leave Management
18. Personal Leave
19. Attendance Management
20. Menu Management
21. Permission Management
22. Role Management
